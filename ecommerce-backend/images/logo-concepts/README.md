# i-emporio Logo Concepts

Generated modern SVG logos for an ecommerce brand.

Files:

- `i-emporio-logo-concept-1.svg`
- `i-emporio-logo-concept-2.svg`
- `i-emporio-logo-concept-3.svg`

Color settings:

- Brand letter color is olive green via `--brand: #6b8e23`.
- Background is fillable via `--bg` in each SVG.

How to customize:

1. Open any SVG file.
2. Change `--bg` for background color.
3. Change `--brand` if you want a different olive shade.

Tip:

- Keep these as SVG for best quality at any size.
